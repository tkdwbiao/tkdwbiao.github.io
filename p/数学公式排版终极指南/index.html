数学公式排版终极指南

本指南涵盖了在 Hugo 博客中使用 KaTeX 排版数学公式的所有常用语法与进阶技巧。

2026-03-04
阅读时长: 2 分钟

在技术博客中，优雅地展示数学公式是必不可少的。本博客使用 KaTeX 引擎进行渲染，它以速度快、排版精美著称。

1. 快速启用

在 Hugo Stack 主题中，你只需要在文章开头的 Front Matter 中添加一行配置即可：

1

math: true

2. 基础语法

行内公式 (Inline Math)

使用单个 $ 符号包裹公式。

  • 输入：这是行内公式：$\alpha + \beta = \gamma$
  • 效果：这是行内公式：$\alpha + \beta = \gamma$

块级公式 (Block Math)

使用两个 $$ 符号包裹公式，公式将独立成行并居中。

4. 进阶排版示例

A. 求和、积分与极限

这些算子在块级模式下会自动变大。

  • 求和：\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
    $\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$

  • 积分：\int_{0}^{\pi} \sin x \, dx = 2
    $\int_{0}^{\pi} \sin x \, dx = 2$

  • 极限：\lim_{x \to \infty} \exp(-x) = 0
    $\lim_{x \to \infty} \exp(-x) = 0$

5. 常见问题排坑

  1. 反斜杠冲突：在 Markdown 环境下，有些字符会被解析引擎拦截。如果你的矩阵 \\ 无法换行，请尝试写成 \\\ (三个反斜杠)。
  2. 特殊字符：大括号 {} 在 LaTeX 中用于分组。如果你想显示大括号本身，需要转义：\{ \}。
  3. 空格：LaTeX 默认忽略空格。如果你需要在公式中插入空格，请使用 \, (小空格) 或 \quad (大空格)。
