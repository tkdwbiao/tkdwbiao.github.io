Python 堆算法库 heapq 全指南：从基础操作到 Top-K 实战

2026-03-20 13:59
阅读时长: 3 分钟

在算法面试中，凡是涉及到“最、前 K 个、动态中位数”等字眼时，堆（Heap / Priority Queue） 就是我们要找的利刃。Python 内置的 heapq 库非常轻量高效，今天阿彪带大家彻底掌握它的常用方法。

1. heapq 常用 API 大全

函数描述时间复杂度
heapq.heapify(list)原地将列表转为最小堆$O(N)$
heapq.heappush(heap, item)向堆中添加一个元素$O(\log N)$
heapq.heappop(heap)弹出并返回最小的元素（堆顶）$O(\log N)$
heapq.heapreplace(heap, item)弹出最小元素，并压入新元素（先弹后压）$O(\log N)$
heapq.heappushpop(heap, item)压入新元素，并弹出最小元素（先压后弹）$O(\log N)$
heapq.nlargest(k, iter)返回迭代器中最大的前 K 个元素$O(N \log K)$
heapq.nsmallest(k, iter)返回迭代器中最小的前 K 个元素$O(N \log K)$

import heapq

# 准备一个无序列表
data = [5, 1, 9, 3, 7]

# 1. 建立堆
heapq.heapify(data) 

# 2. 插入新数据
heapq.heappush(data, 4)

# 3. 循环弹出直到堆空（你会得到一个升序序列）
while data:
    print(heapq.heappop(data)) # 输出: 1, 3, 4, 5, 7, 9

实例 B：大顶堆的“负号魔法”

Python 默认只支持最小堆。如果要实现大顶堆，阿彪教你一招：入堆前取负，出堆后还原。

max_heap = []

# 入堆：变成 [-10, -2, -15, -30]
for x in nums:
    heapq.heappush(max_heap, -x)

# 弹出最大值：结果是 30
biggest = -heapq.heappop(max_heap)

from collections import Counter
import heapq

def topKFrequent(nums, k):
    # 1. 统计频率 {数字: 频率}
    count = Counter(nums)
    
    # 2. 维护一个大小为 k 的最小堆
    # 堆中存储: (频率, 数字)
    hp = []
    for num, freq in count.items():
        if len(hp) < k:
            heapq.heappush(hp, (freq, num))
        else:
            # 如果当前频率比堆顶（目前最小的频率）大
            if freq > hp[0][0]:
                heapq.heapreplace(hp, (freq, num))
    
    # 3. 提取结果
    return [item[1] for item in hp]

# 示例: nums = [1,1,1,2,2,3], k = 2 -> 返回 [1, 2]

3. 阿彪的进阶避坑笔记 [抱拳]

① 为什么 heapify 是 $O(N)$ 而不是 $O(N \log N)$？

面试官如果问起，阿彪你可以这样答：“heapify 并不是通过 $N$ 次插入实现的，它是从最后一个非叶子节点开始，自底向上进行下滤（Sift Down）调整。数学证明其总交换次数收敛于 $O(N)$。”

② 多个字段如何排序？

如果入堆的是元组，如 (priority, name)，heapq 会先比第一个，如果第一个相等，再比第二个。

  • 注意：如果第二个字段（如 name）不支持比较，程序会报错。

③ heapreplace vs heappushpop

  • heapreplace(hp, x)：先弹出堆顶，再压入 x。堆的大小不变。
  • heappushpop(hp, x)：先压入 x，再弹出堆顶。常用于处理 x 还没堆顶大的情况。
  • 性能：这两个复合函数都比拆开写 push 和 pop 要快得多。

4. 阿彪的面试总结

堆在面试中主要解决**数据流（Stream）**中的动态极值问题。

  • 求最大 K 个 $\rightarrow$ 用小顶堆（踢掉弱者，留下强者）。
  • 求最小 K 个 $\rightarrow$ 用大顶堆（踢掉强者，留下弱者）。
  • 求中位数 $\rightarrow$ 用 “对顶堆”（左边大顶堆，右边小顶堆）。
