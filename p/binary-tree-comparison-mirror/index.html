二叉树的“克隆”与“照镜子”：深度拆解相同树与对称树

2026-03-19 10:00
阅读时长: 2 分钟

在二叉树面试题中，比较两棵树的结构是基础中的基础。今天阿彪带大家拆解三份核心代码，看透 相同的树（LeetCode 100）与 对称二叉树（LeetCode 101）背后的底层逻辑。

一、 相同的树：递归的 1:1 复制

判断两棵树是否相同，逻辑非常直观：根节点一样，左子树一样，右子树也一样。

class Solution:
    def isSameTree(self, p: Optional[TreeNode], q: Optional[TreeNode]) -> bool:
        # 1. 终止条件：如果有一个为空，利用 is 判断是否两个都为空
        if p is None or q is None:
            return p is q
        
        # 2. 递归：值相等 且 左边对应 且 右边对应
        return (p.val == q.val and 
                self.isSameTree(p.left, q.left) and 
                self.isSameTree(p.right, q.right))

阿彪的细节笔记：

  • return p is q 是一个非常优雅的写法。如果两个都是 None 返回 True；一有一无则返回 False。
  • 复杂度：时间 $O(n)$，空间 $O(h)$（取决于树高）。

二、 对称二叉树（方法一）：层序遍历与“回文数组”

from collections import deque

class Solution:
    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        # 层序遍历 + 数组反转判断
        queue = deque([root])
        while queue:
            nxt = []    # 存储下一层节点
            value = []  # 存储当前层的值
            for node in queue:
                if node:
                    value.append(node.val)
                    nxt.append(node.left)
                    nxt.append(node.right)
                else:
                    # 关键：必须填入 None 占位，否则无法判断结构对称
                    value.append(None)
            
            # 判断当前层是否是“回文”的
            if value != value[::-1]:
                return False
            
            queue = nxt
        return True

阿彪的深度思考：

“如果没有 None 占位，[1, 2, None] 和 [1, None, 2] 的值数组都是 [1, 2]，这会误判结构！”

这种解法虽然直观，但空间复杂度较高，因为我们要存储大量 `None` 占位符。

三、 对称二叉树（方法二）：镜像递归（最优解）

“对称”其实就是左树和右树互为镜像。我们可以把“相同的树”稍作修改：

  • 比较 p.left 和 q.right（外侧）
  • 比较 p.right 和 q.left（内侧）

class Solution:
    # 这里的逻辑其实是 checkMirror(左树, 右树)
    def isSameTree(self, p: Optional[TreeNode], q: Optional[TreeNode]) -> bool:
        if p is None or q is None:
            return p is q
        
        # 核心变动：左对右，右对左
        return (p.val == q.val and 
                self.isSameTree(p.left, q.right) and 
                self.isSameTree(p.right, q.left))

    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        if not root: return True
        return self.isSameTree(root.left, root.right)

四、 总结：如何选择解法？

维度相同的树 (DFS)对称树 (BFS)对称树 (镜像递归)
思想1:1 匹配逐层回文校验镜像递归匹配
空间复杂度$O(h)$$O(w)$ (w包含空节点)$O(h)$
面试表现基础扎实思路灵活最受面试官推崇

阿彪小结 [抱拳]：

  1. 递归三部曲：先判空（p is q），再比值（p.val == q.val），最后决定递归方向。
  2. BFS 注意点：层序遍历做结构判断时，None 占位符是灵魂。
  3. 镜像思维：对称不是 left 对 left，而是 left 对 right。
