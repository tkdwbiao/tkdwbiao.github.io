阅读时长: 2 分钟

这篇博文旨在展示在 Hugo 博客中，中文长段落的视觉压力以及各种排版元素的呈现效果。通过这篇文章，你可以直观地感受到文字与图片之间的留白是否舒适。

1. 段落与列表测试

一个舒适的阅读体验往往来自于合理的行间距。在这里，我们测试了有序列表的对齐方式和缩进效果：

  1. 第一项：测试列表项的字体加粗效果
  2. 第二项：测试较长文字在多行显示时的缩进
  3. 第三项：检查列表数字的颜色是否与主题一致
  4. 第四项：测试列表与后续段落的间距

在排版中，文字的流动性至关重要。如果我们不间断地书写，段落会呈现出一种块状感，这在浅色模式和暗色模式下都会有不同的视觉表现。

二级标题：视觉平衡与色彩

在设计中，色彩与对比度是吸引读者的关键。通过设置 image 前置参数，你可以在主页看到精美的封面图。此外，加粗的文字（如这段话）能够帮助读者在快速扫视时抓住重点。

文字是思想的载体，而排版是思想的容器。在 Hugo 这种静态站点生成器中，我们可以通过修改 SCSS 文件来微调每一个像素。比如背景色是否过白、字体是否过细，这些细微的调整最终会汇聚成独特的个人风格。

最后，这篇文章还使用了 Hugo 的“短代码”（Shortcode）功能，在文章底部注入了一段局部的 CSS 样式，用于确保上方的 SVG 图形在不同屏幕下都能自适应宽度。
