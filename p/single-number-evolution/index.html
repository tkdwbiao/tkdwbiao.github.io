从暴力到魔法：深度拆解『只出现一次的数字』的四种境界

2026-03-22 22:00
阅读时长: 3 分钟

在算法面试中，有一类题目要求在 $O(n)$ 时间和 $O(1)$ 空间内解决。今天阿彪带大家看的 LeetCode 136. 只出现一次的数字 就是其中的佼佼者。

这道题的背景很简单：一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。我们要找出那个唯一的“异类”。

境界一：哈希表计数（最稳健的思路）

如果不考虑空间复杂度，哈希表是所有“计数”类问题的通解。

核心思想： 遍历数组，记录每个数字出现的次数。最后找出次数为 1 的那个。

def singleNumber(nums):
    count = collections.Counter(nums)
    for k, v in count.items():
        if v == 1:
            return k
  • 时间复杂度：$O(n)$
  • 空间复杂度：$O(n)$。需要存储 $n$ 个元素。

境界二：集合去重（逻辑上的推导）

核心思想： 利用 Set 集合不可重复的特性。遍历数组：

  1. 如果数字不在集合里，存进去。
  2. 如果数字已经在集合里，把它删掉。
  3. 结果：最后剩下的那个就是只出现一次的。

def singleNumber(nums):
    s = set()
    for num in nums:
        if num in s:
            s.remove(num)
        else:
            s.add(num)
    return list(s)[0]
  • 复杂度：时间 $O(n)$，空间 $O(n)$。

境界三：数学转换（降维打击）

这是一套非常“聪明”的解法。

核心思想： 假设数组中唯一存在的数字构成的集合为 $S$。 公式：$2 \times (\text{集合 } S \text{ 之和}) - (\text{原数组之和}) = \text{只出现一次的数字}$

例子：nums = [4, 1, 2, 1, 2]

  • 集合 $S = \{1, 2, 4\}$，和为 7。
  • 原数组和 = $1+1+2+2+4 = 10$。
  • $2 \times 7 - 10 = 4$。完美！
1
2

def singleNumber(nums):
    return 2 * sum(set(nums)) - sum(nums)
  • 复杂度：时间 $O(n)$，空间 $O(n)$（由 set 产生的空间）。

境界四：异或魔法（极致 $O(1)$ 空间）

这是阿彪最推崇的解法，也是面试官要求的最优解。

核心思想： 利用二进制运算符 异或（XOR） 的三个神奇性质：

  1. 归零律：$a \oplus a = 0$（任何数和自己异或，结果为 0）
  2. 恒等律：$a \oplus 0 = a$（任何数和 0 异或，结果还是自己）
  3. 交换律/结合律：$a \oplus b \oplus a = (a \oplus a) \oplus b = 0 \oplus b = b$

结论：我们将数组里所有的数字全部异或一遍，那些成双成对的数字会在异或运算中两两抵消变成 0，最后剩下的那个数字就是我们要找的答案。

class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        res = 0
        for num in nums:
            # 不断累计异或结果
            res ^= num
        return res

阿彪的总结对比表

解法时间复杂度空间复杂度评价
哈希表$O(n)$$O(n)$基础扎实，易于扩展（比如找出现三次的）
集合法$O(n)$$O(n)$逻辑清晰，代码简洁
数学法$O(n)$$O(n)$思路清奇，展示数学功底
异或法$O(n)$$O(1)$面试标准答案，满分表现

阿彪的面试金句 [抱拳]

“面试官，这道题的直观解法是使用哈希表进行频次统计，但这需要 $O(n)$ 的额外空间。考虑到题目中‘成对出现’的特殊约束，我采用了基于异或运算（XOR）的位运算解法。由于异或运算满足交换律且相同数字异或为 0，我们可以通过一次遍历抵消掉所有重复数字，在 $O(1)$ 空间内精准锁定目标。”

位运算是算法题中的“黑魔法”，掌握了异或，你就掌握了这类‘找异类’问题的万能钥匙！加油阿彪！

阿彪的建议：

  1. 关于异或的物理理解：你可以把异或想象成一个 “开关”。按两次开关，灯回到了原来的状态（熄灭/0）；按一次开关，灯就亮了（保留）。所有的成对数字都在开关灯，最后亮着的那盏灯就是答案。
  2. 扩展题目：如果面试官追问：“如果有两个数字只出现了一次，剩下的都出现了两次，怎么做？” 这是一个经典的 Hard 题（LC 260），你可以去搜一下，也是用异或解决的，非常精彩。
