Markdown 语法全面指南

本文展示了在 Hugo 中使用的基本 Markdown 语法，以及各种 HTML 元素在主题中的渲染效果。

2026-03-06
阅读时长: 2 分钟

这篇文章为你提供了一个基础 Markdown 语法的示例，你可以参考这些语法在 Hugo 中撰写内容。同时，它也展示了基本 HTML 元素在当前主题下的 CSS 装饰效果。

1. 标题 (Headings)

HTML 中的 <h1> 到 <h6> 元素代表了六个级别的节标题。<h1> 是最高级别，而 <h6> 是最低级别。

一级标题 (H1)

二级标题 (H2)

三级标题 (H3)

四级标题 (H4)

五级标题 (H5)
六级标题 (H6)

2. 段落 (Paragraph)

这是一个标准的段落。Markdown 允许你轻松地书写文字。在 Hugo 中，段落会被自动包裹在 HTML 的 <p> 标签中。

如果你想换行，可以在行尾输入两个空格，或者直接空一行开启一个新段落。博文的排版应该保持简洁明了，适当的留白会让读者的阅读体验更加舒适。

3. 引用 (Blockquotes)

引用元素代表摘自另一个来源的内容。你可以选择性地添加来源说明。

不带署名的引用

这是一个引用块。Markdown 的语法在引用块内部依然有效。 注意：你可以在引用中使用 斜体 或 加粗。

带署名的引用

不要通过共享内存来通信，而要通过通信来共享内存。
— Rob Pike1
