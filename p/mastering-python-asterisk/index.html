Python 星号 (*) 全攻略：从零到深层解包的魔法

2026-03-23 20:00
阅读时长: 3 分钟

在 Python 中，星号 * 不仅仅是乘法运算符。它出现在函数参数、变量赋值、甚至是矩阵操作中。今天阿彪带大家彻底盘点 * 和 ** 的所有神奇用法。

1. 基础数学运算

这是最入门的用法，大家都懂，简单带过：

  • 单星号 *：乘法。
  • 双星号 **：幂运算（次方）。
1
2

print(2 * 3)   # 6
print(2 ** 3)  # 8 (2的3次方)

2. 容器展开（解包 Unpacking）—— 面试高频！

这是 * 最迷人的地方。它能把容器里的元素“倒”出来。

① 在列表/元组中使用

1
2
3
4
5
6
7

nums = [1, 2, 3]
print(*nums)  # 等同于 print(1, 2, 3) -> 输出: 1 2 3

# 合并列表
list1 = [1, 2]
list2 = [3, 4]
merged = [*list1, *list2] # [1, 2, 3, 4]

② 矩阵转置的黑科技

还记得我们在《矩阵置零》中用的 zip(*matrix) 吗？

1
2
3
4

matrix = [[1, 2], [3, 4]]
# *matrix 会把两行拆开：[1, 2] 和 [3, 4]
# zip 再把对应的竖着打包
transposed = list(zip(*matrix)) # [(1, 3), (2, 4)]

3. 函数参数的“变色龙”

① *args：接收任意数量的位置参数

当你不知道用户会传多少个参数时使用。它会将多余的参数打包成一个元组（Tuple）。

1
2
3
4

def add_all(*args):
    return sum(args)

print(add_all(1, 2, 3, 4)) # args 为 (1, 2, 3, 4)

② **kwargs：接收任意数量的关键字参数

它会将传进来的键值对打包成一个字典（Dict）。

1
2
3
4
5

def info(**kwargs):
    for k, v in kwargs.items():
        print(f"{k}: {v}")

info(name="阿彪", goal="Offer") # kwargs 为 {'name': '阿彪', 'goal': 'Offer'}

4. 强制关键字参数（Naked Asterisk）

这是 Python 3 引入的高级特性。如果你想强制调用者必须写出参数名，就在参数列表里放一个孤独的 *。

1
2
3
4
5

def connect(*, host, port):
    pass

# connect("127.0.0.1", 8080)   # ❌ 报错
connect(host="127.0.0.1", port=8080) # ✅ 正确

阿彪点睛：这在设计大型库时非常有用，能显著提高代码的可读性，防止调号传错参数。

5. 变量赋值中的“贪婪捕获”

在解构赋值时，可以用 * 拿走剩下的所有东西。

1
2
3
4
5
6
7
8

a, b, *others = [1, 2, 3, 4, 5]
print(a)      # 1
print(b)      # 2
print(others) # [3, 4, 5]

# 甚至可以放中间
first, *middle, last = [10, 20, 30, 40]
print(middle) # [20, 30]

6. 字典的合并与解包 (**)

双星号不仅用于函数参数，还能合并字典。

1
2
3

dict_a = {'x': 1, 'y': 2}
dict_b = {'z': 3}
merged_dict = {**dict_a, **dict_b} # {'x': 1, 'y': 2, 'z': 3}

7. 重复序列

快速生成带有重复初始值的列表或字符串。

1
2

zeros = [0] * 5    # [0, 0, 0, 0, 0]
line = "-" * 10    # "----------"

阿彪的面试避坑笔记 [抱拳]

面试官追问：*args 和 **kwargs 同时出现怎么办？

阿彪回答：它们的顺序必须是固定的。标准顺序是：func(普通参数, *args, **kwargs)。这是因为位置参数必须在关键字参数之前。

面试官追问：* 在解包时是浅拷贝还是深拷贝？

阿彪回答：是 浅拷贝。如果是嵌套列表，解包出来的元素依然指向原内存地址。

总结表

用法语法效果
解包*iter将容器里的东西倒出来
打包def f(*args)将散乱的参数装进元组
字典解包**dict处理键值对
捕获剩下a, *b = lst把多余的都给 b
强制命名def f(*, a)a 必须指名道姓地传
